📌 Когда применяется метод координат

  • Задачи на нахождение минимального расстояния между точками
  • Задачи на экстремум функции, связанной с расстоянием

⚙️ Алгоритм решения

Шаг 1. Запись функции расстояния

Пусть точка A(x₀;y₀) и точка на кривой B(t; f(t))
Квадрат расстояния: d²(t) = (t − x₀)² + (f(t) − y₀)²

Шаг 2. Исследование функции на минимум

Находим производную (d²)'(t)
Приравниваем к нулю: (d²)'(t) = 0
Находим критические точки

Шаг 3. Вычисление минимального значения

Подставляем найденные t в функцию расстояния
Выбираем наименьшее значение

Шаг 4. Для функций на отрезке

Находим значения на концах отрезка
Сравниваем со значениями в критических точках

📖 Пример

Условие: Из точки A(0;5) на параболу y = x²/4 бросили мяч. Найдите абсциссу точки приземления M, при которой расстояние AM будет наименьшим.

Решение:
M(t; t²/4)
d²(t) = t² + (t²/4 − 5)² = t² + t⁴/16 − 5t²/2 + 25 = t⁴/16 − 3t²/2 + 25
(d²)' = t³/4 − 3t = t(t²/4 − 3) = 0 ⇒ t = 2√3
Ответ: 2√3
