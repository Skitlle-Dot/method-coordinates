📌 Когда применяется метод координат

  • Неравенства с двумя переменными
  • Системы неравенств, задающие области на плоскости
  • Задачи с параметрами, имеющие геометрический смысл

⚙️ Алгоритм решения

Шаг 1. Геометрическая интерпретация

Представляем каждое неравенство как условие на координаты точек.

Шаг 2. Построение границ

Прямая: Ax + By + C = 0
Окружность: (x − x₀)² + (y − y₀)² = R²
Парабола: y = ax² + bx + c
Гипербола: xy = k

Шаг 3. Определение области

Для каждого неравенства определяем, какая часть плоскости удовлетворяет условию.

Для прямой: подставляем точку (0;0) или другую удобную точку
Для кривой: анализируем знак выражения

Шаг 4. Нахождение пересечения

Решение системы — пересечение всех областей. Находим точки пересечения границ.

Шаг 5. Запись ответа

Ответ записываем в виде области или в аналитической форме.

📖 Пример

Условие: Изобразите множество точек, удовлетворяющих неравенству x² + (y − 1)² ≤ 4.

Решение:
Это круг с центром C(0;1) и радиусом R = 2. Неравенство нестрогое, включаем границу.

Ответ: Круг радиусом 2 с центром (0;1), включая границу.
