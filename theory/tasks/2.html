📌 Когда применяется метод координат

  • Любые задачи с векторами, где координаты даны или легко находятся
  • Нахождение скалярного произведения
  • Определение угла между векторами
  • Проверка коллинеарности и перпендикулярности

⚙️ Алгоритм решения

Шаг 1. Нахождение координат векторов

AB = {x_B − x_A; y_B − y_A}
В пространстве: AB = {x_B − x_A; y_B − y_A; z_B − z_A}

Шаг 2. Вычисление длины вектора (модуля)

|a| = √(x² + y²) — на плоскости
|a| = √(x² + y² + z²) — в пространстве

Шаг 3. Скалярное произведение

a·b = |a|·|b|·cos(a,b)
a·b = x₁·x₂ + y₁·y₂ (на плоскости)
a·b = x₁·x₂ + y₁·y₂ + z₁·z₂ (в пространстве)

Шаг 4. Угол между векторами

cos(a,b) = (a·b) / (|a|·|b|)

Шаг 5. Условия

Перпендикулярность: a·b = 0
Коллинеарность: x₁/x₂ = y₁/y₂ (на плоскости)
Коллинеарность: x₁/x₂ = y₁/y₂ = z₁/z₂ (в пространстве)

📖 Пример

Условие: Даны векторы a(3;4) и b(6;8). Найдите угол между ними.

Решение:
a·b = 3·6 + 4·8 = 18 + 32 = 50
|a| = √(9+16) = 5, |b| = √(36+64) = 10
cos φ = 50/(5·10) = 1 ⇒ φ = 0° (векторы коллинеарны)

Ответ: 0°
