📌 Когда применяется метод координат

  • Задачи с многоугольниками, где можно ввести систему координат
  • Доказательство свойств фигур
  • Нахождение геометрических мест точек

⚙️ Алгоритм решения

Шаг 1. Введение системы координат

  • Используем симметрию фигуры
  • Помещаем начало в удобную точку (вершину, центр симметрии)
  • Оси направляем по осям симметрии или сторонам

Шаг 2. Запись координат вершин

Для прямоугольных фигур: координаты через длины сторон
Для ромба: A(0;a), B(b;0), C(0;-a), D(-b;0)
Для равнобедренного треугольника: A(0;0), B(c;0), C(c/2; h)

Шаг 3. Выражение условия задачи

Записываем в координатной форме то, что нужно доказать или найти:

Перпендикулярность: (вектор1)·(вектор2) = 0
Равенство длин: |AB| = |CD|
Принадлежность точки прямой: уравнение прямой

Шаг 4. Алгебраические преобразования

Раскрываем скобки, упрощаем выражения. Используем свойства координат.

Шаг 5. Получение результата

Делаем вывод из полученных равенств или неравенств.

📖 Пример

Условие: В квадрате ABCD точки M и N — середины сторон AB и BC. Докажите, что CM ⟂ DN.

Решение:
A(0;0), B(1;0), C(1;1), D(0;1), M(0,5;0), N(1;0,5)
CM = M − C = (-0,5; -1), DN = N − D = (1; -0,5)
CM·DN = (-0,5)·1 + (-1)·(-0,5) = -0,5 + 0,5 = 0 ⇒ CM ⟂ DN

Ответ: Доказано.
