📌 Когда применяется метод координат

  • Задачи на движение по плоскости
  • Задачи с координатами движущихся объектов
  • Нахождение минимального расстояния между объектами

⚙️ Алгоритм решения

Шаг 1. Введение системы координат

Помещаем начальные точки в удобные координаты. Определяем направления движения.

Шаг 2. Параметрические уравнения движения

x₁(t) = x₁₀ + v₁ₓ·t, y₁(t) = y₁₀ + v₁ᵧ·t
x₂(t) = x₂₀ + v₂ₓ·t, y₂(t) = y₂₀ + v₂ᵧ·t

Шаг 3. Расстояние между объектами

d²(t) = (x₁(t) − x₂(t))² + (y₁(t) − y₂(t))²

Раскрываем скобки, получаем квадратичную функцию от t.

Шаг 4. Исследование на минимум

Для квадратичной функции: минимум при t = −b/(2a) (если a > 0). Подставляем найденное t в формулу расстояния.

Для неквадратичных функций: находим производную, приравниваем к нулю, находим критические точки.

📖 Пример

Условие: Два автомобиля выезжают из одной точки: первый на север со скоростью 40 км/ч, второй на восток со скоростью 30 км/ч. Найдите расстояние между ними через t часов.

Решение:
x₁=0, y₁=40t, x₂=30t, y₂=0
d² = (0−30t)² + (40t−0)² = 900t² + 1600t² = 2500t² ⇒ d = 50t

Ответ: 50t км
