📌 Когда применяется метод координат

  • Угол между прямыми
  • Угол между прямой и плоскостью
  • Угол между плоскостями
  • Расстояние от точки до плоскости
  • Расстояние между скрещивающимися прямыми

⚙️ Алгоритм решения

Шаг 1. Введение системы координат

Выбираем удобное расположение (как в задании 3).

Шаг 2. Запись координат всех точек

Находим координаты всех упомянутых точек.

Шаг 3. Угол между прямыми

cos φ = |p·q| / (|p|·|q|)

Шаг 4. Угол между прямой и плоскостью

sin φ = |a·n| / (|a|·|n|)

Шаг 5. Угол между плоскостями

cos φ = |n₁·n₂| / (|n₁|·|n₂|)

Шаг 6. Расстояние от точки до плоскости

Составляем уравнение плоскости Ax + By + Cz + D = 0
d = |Ax₀ + By₀ + Cz₀ + D| / √(A² + B² + C²)

Шаг 7. Расстояние между скрещивающимися прямыми

ρ = |(a·[b×c])| / |a×b|

📖 Пример

Условие: В кубе ABCDA₁B₁C₁D₁ с ребром 1 найдите расстояние между прямыми AB₁ и BC₁.

Решение:
A(0;0;0), B₁(1;0;1), B(1;0;0), C₁(1;1;1)
p = AB₁ = {1;0;1}, q = BC₁ = {0;1;1}, c = AB = {1;0;0}
Смешанное произведение = 1, |p×q| = √3 ⇒ d = 1/√3 = √3/3

Ответ: √3/3
