📌 Когда применяется метод координат

  • Треугольники с заданными координатами вершин
  • Параллелограммы, ромбы, трапеции
  • Окружности и их элементы
  • Задачи на нахождение расстояний, углов, площадей

⚙️ Алгоритм решения

Шаг 1. Введение системы координат

  • Для прямоугольных фигур — начало в вершине, оси по сторонам
  • Для ромба — оси по диагоналям
  • Для окружности — начало в центре
  • Для трапеции — начало в нижнем левом углу, ось X по основанию

Шаг 2. Запись координат точек

Середина отрезка: \(\displaystyle x = \frac{x_1 + x_2}{2},\; y = \frac{y_1 + y_2}{2}\)
Деление отрезка в отношении \(\lambda\): \(\displaystyle x = \frac{x_1 + \lambda x_2}{1 + \lambda},\; y = \frac{y_1 + \lambda y_2}{1 + \lambda}\)

Шаг 3. Применение формул

Расстояние между точками: \(\displaystyle d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}\)
Площадь треугольника: \(\displaystyle S = \frac{1}{2} \left| (x_2 - x_1)(y_3 - y_1) - (x_3 - x_1)(y_2 - y_1) \right|\)
Уравнение окружности: \((x - x_0)^2 + (y - y_0)^2 = R^2\)
Координаты центра тяжести треугольника: \(\displaystyle x = \frac{x_1 + x_2 + x_3}{3},\; y = \frac{y_1 + y_2 + y_3}{3}\)

Шаг 4. Вычисление и ответ

Подставляем числа, упрощаем, записываем ответ.

📖 Пример

Условие: Треугольник ABC с вершинами \(A(1;2), B(5;6), C(3;8)\). Найдите длину медианы \(AM\).

Решение:
\(M\) — середина \(BC\): \(\displaystyle M\left(\frac{5+3}{2}; \frac{6+8}{2}\right) = (4;7)\)
\(\displaystyle AM = \sqrt{(4-1)^2 + (7-2)^2} = \sqrt{9 + 25} = \sqrt{34}\)

Ответ: \(\sqrt{34}\)
