📖 Условие

В прямоугольном параллелепипеде ABCDA₁B₁C₁D₁ известны ребра: AB = 3, AD = 4, AA₁ = 6. Найдите угол между прямой CD₁ и плоскостью B₁BD.

✅ Ответ

\(\arcsin \dfrac{4}{5\sqrt{5}}\)

📖 Подробный разбор

1. Вводим систему координат: A(0;0;0), B(3;0;0), D(0;4;0), A₁(0;0;6), B₁(3;0;6), C(3;4;0), D₁(0;4;6)

2. Направляющий вектор прямой CD₁:
\(\overrightarrow{CD₁} = (0-3; 4-4; 6-0) = (-3; 0; 6)\)

3. Находим нормаль к плоскости B₁BD через точки B₁(3;0;6), B(3;0;0), D(0;4;0):
\(\overrightarrow{BB₁} = (0; 0; 6)\)
\(\overrightarrow{BD} = (-3; 4; 0)\)
\(\vec{n} = \overrightarrow{BB₁} × \overrightarrow{BD} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 0 & 0 & 6 \\ -3 & 4 & 0 \end{vmatrix} = (-24; -18; 0) = -6·(4; 3; 0)\)
Принимаем \(\vec{n} = (4; 3; 0)\)

4. Синус угла:
\(\sin φ = \dfrac{|\overrightarrow{CD₁}·\vec{n}|}{|\overrightarrow{CD₁}|·|\vec{n}|} = \dfrac{|(-3)·4 + 0·3 + 6·0|}{√(9 + 0 + 36)·√(16 + 9 + 0)} = \dfrac{12}{√45·5} = \dfrac{12}{5√45} = \dfrac{12}{15√5} = \dfrac{4}{5√5}\)

5. Угол:
\(φ = \arcsin \dfrac{4}{5√5}\)
