📖 Условие

Из точки \(A(0;5)\) на параболу \(y = \dfrac{x^2}{4}\) бросили мяч. Найдите абсциссу точки приземления \(M\), при которой длина траектории (расстояние \(AM\)) будет наименьшей.

✅ Ответ

\(2\sqrt{3}\)

📖 Подробный разбор

1. Пусть точка \(M\) имеет координаты \(\left(t; \dfrac{t^2}{4}\right)\)

2. Квадрат расстояния между \(A\) и \(M\):
\(d^2(t) = (t - 0)^2 + \left(\dfrac{t^2}{4} - 5\right)^2 = t^2 + \dfrac{t^4}{16} - \dfrac{5t^2}{2} + 25 = \dfrac{t^4}{16} - \dfrac{3t^2}{2} + 25\)

3. Находим производную:
\(f'(t) = \dfrac{4t^3}{16} - 3t = \dfrac{t^3}{4} - 3t = t\left(\dfrac{t^2}{4} - 3\right)\)

4. Приравниваем к нулю:
\(t = 0\) или \(\dfrac{t^2}{4} = 3 \Rightarrow t^2 = 12 \Rightarrow t = \pm 2\sqrt{3}\)

5. Вычисляем квадрат расстояния для \(t = 2\sqrt{3}\):
\(d^2 = \dfrac{(2\sqrt{3})^4}{16} - \dfrac{3\cdot(2\sqrt{3})^2}{2} + 25 = \dfrac{144}{16} - \dfrac{36}{2} + 25 = 9 - 18 + 25 = 16\)
Для \(t = 0\): \(d^2 = 25\)

6. Минимум достигается при \(t = 2\sqrt{3}\)
