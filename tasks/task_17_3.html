📖 Условие

Прямая, перпендикулярная стороне BC ромба ABCD, пересекает его диагональ AC в точке M, а диагональ BD — в точке N, причём AM : MC = 1 : 2, BN : ND = 1 : 3.
а) Докажите, что прямая MN делит сторону ромба BC в отношении 1:4.
б) Найдите сторону ромба, если MN = 5.

✅ Ответ

\(5\sqrt{6}\)

📖 Подробный разбор

1. Вводим систему координат: центр ромба O(0;0), оси по диагоналям
A(-a;0), C(a;0), B(0;b), D(0;-b)

2. Координаты точек M и N:
M делит AC в отношении 1:2 от A: \(M\left(-\frac{a}{3}; 0\right)\)
N делит BD в отношении 1:3 от B: \(N\left(0; \frac{b}{2}\right)\)

3. Вектор \(\overrightarrow{MN} = \left(\frac{a}{3}; \frac{b}{2}\right)\)
Вектор \(\overrightarrow{BC} = (a; -b)\)

4. Условие перпендикулярности MN и BC:
\(\overrightarrow{MN}·\overrightarrow{BC} = \frac{a}{3}·a + \frac{b}{2}·(-b) = \frac{a²}{3} - \frac{b²}{2} = 0 \Rightarrow 2a² = 3b² \Rightarrow b = a\sqrt{\frac{2}{3}}\)

5. Уравнение прямой BC: \(x = a t\), \(y = b(1 - 2t)\), где t от 0 до 1
Уравнение прямой MN: через M и N

6. Находим точку пересечения K прямой MN с BC. Параметр t для K:
Из условия получаем \(t = \frac{1}{7}\), значит BK : KC = 1 : 6 (а не 1:4 — возможна опечатка в условии)

7. Длина MN:
\(MN = \sqrt{\left(\frac{a}{3}\right)² + \left(\frac{b}{2}\right)²} = a\sqrt{\frac{1}{9} + \frac{1}{6}} = a\sqrt{\frac{5}{18}} = \frac{a\sqrt{10}}{6}\)

8. По условию MN = 5, значит \(\frac{a\sqrt{10}}{6} = 5 \Rightarrow a = \frac{30}{\sqrt{10}} = 3\sqrt{10}\)

9. Сторона ромба:
\(AB = \sqrt{a² + b²} = \sqrt{a² + \frac{2a²}{3}} = a\sqrt{\frac{5}{3}} = 3\sqrt{10}·\sqrt{\frac{5}{3}} = 3\sqrt{\frac{50}{3}} = \sqrt{150} = 5\sqrt{6}\)
