📖 Условие

Из пункта A выехал автомобиль со скоростью 60 км/ч. Через 2 часа из пункта A в том же направлении выехал второй автомобиль со скоростью 80 км/ч. Через какое время после выезда второго автомобиля расстояние между ними будет наименьшим?

✅ Ответ

6 часов

📖 Подробный разбор

1. Вводим систему координат: начало в A, ось X по направлению движения

2. Уравнения движения:
Первый автомобиль: \(x_1(t) = 60(t + 2)\) для \(t \ge 0\) (t отсчитываем от момента выезда второго)
Второй автомобиль: \(x_2(t) = 80t\)

3. Расстояние между ними: \(d(t) = |x_1(t) - x_2(t)| = |60t + 120 - 80t| = |120 - 20t|\)

4. Минимальное расстояние достигается при \(120 - 20t = 0\), то есть \(t = 6\) часов

5. При \(t = 6\): \(d = 0\) (второй догнал первого)
