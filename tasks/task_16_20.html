Задача по методу координат

📖 Условие

Два корабля выходят из порта одновременно. Первый идёт на север со скоростью 30 км/ч, второй — на восток со скоростью 40 км/ч. Через 2 часа первый поворачивает на восток и идёт с той же скоростью. Найдите минимальное расстояние между кораблями после момента поворота.

✅ Ответ

100 км

📖 Подробный разбор

1. Вводим систему координат: порт в начале координат, ось X на восток, ось Y на север

2. До поворота (первые 2 часа):
Первый: \((0; 30t)\) при \(t \le 2\)
Второй: \((40t; 0)\)

3. В момент \(t = 2\):
Первый: \((0; 60)\)
Второй: \((80; 0)\)

4. После поворота (для \(t \ge 2\) вводим новое время \(\tau = t - 2 \ge 0\)):
Первый начинает из \((0; 60)\) и движется на восток со скоростью 30 км/ч:
\(x_1(\tau) = 30\tau\), \(y_1(\tau) = 60\)
Второй продолжает движение на восток:
\(x_2(\tau) = 80 + 40\tau\), \(y_2(\tau) = 0\)

5. Квадрат расстояния:
\(\Delta x = 30\tau - (80 + 40\tau) = -80 - 10\tau\)
\(\Delta y = 60 - 0 = 60\)
\(d^2(\tau) = (80 + 10\tau)^2 + 60^2 = 6400 + 1600\tau + 100\tau^2 + 3600 = 100\tau^2 + 1600\tau + 10000\)

6. Вершина параболы при \(\tau_{min} = -\dfrac{1600}{2 \cdot 100} = -8\) — отрицательное, значит минимум при \(\tau = 0\)

7. Минимальное расстояние после поворота:
\(d_{min} = \sqrt{6400 + 3600} = \sqrt{10000} = 100\) км
