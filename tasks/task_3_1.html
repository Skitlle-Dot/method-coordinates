📖 Условие

В кубе ABCDA₁B₁C₁D₁ найдите угол между прямыми AB₁ и BC₁. Ребро куба принять за 1.

✅ Ответ

60

📖 Подробный разбор

1. Вводим систему координат: A(0;0;0), B(1;0;0), C(1;1;0), D(0;1;0), A₁(0;0;1), B₁(1;0;1), C₁(1;1;1), D₁(0;1;1)

2. Находим координаты векторов:
\(\overrightarrow{AB₁} = (1-0; 0-0; 1-0) = (1; 0; 1)\)
\(\overrightarrow{BC₁} = (1-1; 1-0; 1-0) = (0; 1; 1)\)

3. Скалярное произведение:
\(\overrightarrow{AB₁}·\overrightarrow{BC₁} = 1·0 + 0·1 + 1·1 = 1\)

4. Длины векторов:
\(|\overrightarrow{AB₁}| = √(1² + 0² + 1²) = √2\)
\(|\overrightarrow{BC₁}| = √(0² + 1² + 1²) = √2\)

5. Косинус угла:
\(\cos φ = \dfrac{1}{√2·√2} = \dfrac{1}{2}\)

6. Угол:
\(φ = 60°\)
