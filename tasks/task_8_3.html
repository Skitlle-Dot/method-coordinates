📖 Условие

Прямая \(y = 9x + 6\) является касательной к графику \(y = ax² - 19x + 13\). Найдите \(a\).

✅ Ответ

28

📖 Подробный разбор

1. Производная функции: \(f'(x) = 2ax - 19\)

2. В точке касания \(x₀\):
\(2ax₀ - 19 = 9\) и \(ax₀² - 19x₀ + 13 = 9x₀ + 6\)

3. Из первого уравнения: \(2ax₀ = 28 \Rightarrow ax₀ = 14\)

4. Второе уравнение упрощаем: \(ax₀² - 28x₀ + 7 = 0\)
Подставляем \(a = 14/x₀\):
\(\dfrac{14}{x₀}·x₀² - 28x₀ + 7 = 0\)
\(14x₀ - 28x₀ + 7 = 0\)
\(-14x₀ = -7\)
\(x₀ = 0,5\)

5. Находим \(a = \dfrac{14}{0,5} = 28\)
