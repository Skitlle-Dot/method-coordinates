📖 Условие

Два тела движутся по плоскости. Первое тело начинает движение из точки (0; 0) со скоростью \(v_1 = 5\) м/с вдоль оси X. Второе тело начинает движение из точки (10; 20) со скоростью \(v_2 = 3\) м/с под углом 120° к оси X. Найдите минимальное расстояние между телами в процессе движения.

✅ Ответ

\(\dfrac{1}{7}\sqrt{17575 + 3900\sqrt{3}}\)

📖 Подробный разбор

1. Уравнения движения:
Тело 1: \(x_1(t) = 5t\), \(y_1(t) = 0\)
Тело 2: \(x_2(t) = 10 - 1,5t\), \(y_2(t) = 20 + 1,5\sqrt{3}t\)

2. Квадрат расстояния:
\(\Delta x = 5t - 10 + 1,5t = 6,5t - 10\)
\(\Delta y = 0 - 20 - 1,5\sqrt{3}t = -20 - 1,5\sqrt{3}t\)
\(d^2(t) = (6,5t - 10)^2 + (20 + 1,5\sqrt{3}t)^2 = 49t^2 + (60\sqrt{3} - 130)t + 500\)

3. Вершина параболы при \(t_{min} = \dfrac{130 - 60\sqrt{3}}{98}\)

4. Минимальное расстояние:
\(d_{min} = \dfrac{1}{7}\sqrt{17575 + 3900\sqrt{3}}\)
