📖 Условие

Прямая \(y = -7x + 141\) является касательной к графику функции \(y = x³ + 5x² - 4x + 132\). Найдите ординату точки касания.

✅ Ответ

162

📖 Подробный разбор

1. Производная функции: \(f'(x) = 3x² + 10x - 4\)

2. В точке касания угловой коэффициент прямой равен значению производной:
\(3x₀² + 10x₀ - 4 = -7\)

3. Решаем уравнение:
\(3x₀² + 10x₀ + 3 = 0\)
Дискриминант \(D = 100 - 36 = 64\)
\(x₀ = \dfrac{-10 \pm 8}{6}\)

4. Корни: \(x₁ = -\dfrac{1}{3}\), \(x₂ = -3\)

5. Ордината точки касания — значение функции в этой точке:
Для \(x₀ = -3\): \(y = -7·(-3) + 141 = 21 + 141 = 162\)
Проверка: \(f(-3) = -27 + 45 + 12 + 132 = 162\) — совпадает
