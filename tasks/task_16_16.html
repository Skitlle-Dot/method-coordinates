📖 Условие

Два поезда движутся по прямолинейным пересекающимся путям. Первый поезд в момент времени t = 0 находится на перекрёстке и движется на север со скоростью 50 км/ч. Второй поезд в момент t = 0 находится в 20 км к востоку от перекрёстка и движется на запад со скоростью 30 км/ч. Найдите минимальное расстояние между поездами.

✅ Ответ

\(\dfrac{50\sqrt{34}}{17}\) км

📖 Подробный разбор

1. Вводим систему координат: начало на перекрёстке, ось X на восток, ось Y на север

2. Уравнения движения:
Первый поезд: \(x_1(t) = 0\), \(y_1(t) = 50t\)
Второй поезд: \(x_2(t) = 20 - 30t\), \(y_2(t) = 0\)

3. Квадрат расстояния:
\(\Delta x = 0 - (20 - 30t) = -20 + 30t\)
\(\Delta y = 50t - 0 = 50t\)
\(d^2(t) = (30t - 20)^2 + (50t)^2 = 900t^2 - 1200t + 400 + 2500t^2 = 3400t^2 - 1200t + 400\)

4. Вершина параболы при \(t_{min} = \dfrac{1200}{2 \cdot 3400} = \dfrac{1200}{6800} = \dfrac{12}{68} = \dfrac{3}{17}\) часа ≈ 0,176 ч

5. Минимальное расстояние:
\(d_{min}^2 = 3400 \cdot \left(\dfrac{3}{17}\right)^2 - 1200 \cdot \dfrac{3}{17} + 400 = 3400 \cdot \dfrac{9}{289} - \dfrac{3600}{17} + 400\)
\(d_{min}^2 = \dfrac{30600}{289} - \dfrac{3600}{17} + 400 = \dfrac{30600}{289} - \dfrac{3600 \cdot 17}{289} + \dfrac{400 \cdot 289}{289}\)
\(d_{min}^2 = \dfrac{30600 - 61200 + 115600}{289} = \dfrac{85000}{289}\)
\(d_{min} = \sqrt{\dfrac{85000}{289}} = \dfrac{100\sqrt{8,5}}{17} = \dfrac{100\sqrt{17/2}}{17} = \dfrac{100\sqrt{17}}{17\sqrt{2}} = \dfrac{100\sqrt{34}}{34} = \dfrac{50\sqrt{34}}{17}\) км
