📖 Условие

Из точки \(A(0;2)\) проведён отрезок к точке \(B\) на параболе \(y = x^2\). Длина этого отрезка равна 5. Найдите все возможные координаты точки \(B\).

✅ Ответ

\(\left(\pm \sqrt{\dfrac{3 + \sqrt{93}}{2}} ; \dfrac{3 + \sqrt{93}}{2}\right)\)

📖 Подробный разбор

1. Пусть \(B(t; t^2)\)

2. Квадрат расстояния \(AB\) равен 25:
\(t^2 + (t^2 - 2)^2 = 25\)
\(t^2 + t^4 - 4t^2 + 4 = 25\)
\(t^4 - 3t^2 - 21 = 0\)

3. Замена \(u = t^2 \ge 0\):
\(u^2 - 3u - 21 = 0\)
\(u = \dfrac{3 \pm \sqrt{9 + 84}}{2} = \dfrac{3 \pm \sqrt{93}}{2}\)

4. Корень \(\dfrac{3 - \sqrt{93}}{2}\) отрицательный, не подходит

5. \(u = \dfrac{3 + \sqrt{93}}{2}\), тогда \(t = \pm \sqrt{\dfrac{3 + \sqrt{93}}{2}}\)

6. Координаты точки \(B\):
\(x = \pm \sqrt{\dfrac{3 + \sqrt{93}}{2}}\), \(y = \dfrac{3 + \sqrt{93}}{2}\)
