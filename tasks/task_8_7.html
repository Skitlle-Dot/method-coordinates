📖 Условие

Найдите тангенс угла, который образует с положительным направлением оси абсцисс касательная, проведенная к графику функции \(f(x) = \dfrac{x+3}{x-3}\) в точке \(x₀ = -7\) этого графика.

✅ Ответ

-0,06

📖 Подробный разбор

1. Производная функции (производная частного):
\(f'(x) = \dfrac{1·(x-3) - (x+3)·1}{(x-3)²} = \dfrac{x-3 - x - 3}{(x-3)²} = \dfrac{-6}{(x-3)²}\)

2. Значение производной в точке \(x₀ = -7\):
\(f'(-7) = \dfrac{-6}{(-7-3)²} = \dfrac{-6}{(-10)²} = \dfrac{-6}{100} = -0,06\)

3. Тангенс угла наклона касательной равен значению производной
