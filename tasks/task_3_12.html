📖 Условие

В правильной треугольной призме ABCA₁B₁C₁ все ребра равны 2. Найдите расстояние от точки A до плоскости A₁BC.

✅ Ответ

\(2\sqrt{\dfrac{3}{7}}\)

📖 Подробный разбор

1. Вводим систему координат: A(0;0;0), B(2;0;0), C(1; √3; 0), A₁(0;0;2)

2. Составляем уравнение плоскости A₁BC через точки A₁(0;0;2), B(2;0;0), C(1; √3; 0):
\(\overrightarrow{A₁B} = (2; 0; -2)\)
\(\overrightarrow{A₁C} = (1; √3; -2)\)
Нормаль: \(\vec{n} = \overrightarrow{A₁B} × \overrightarrow{A₁C} = (2√3; 2; 2√3) = 2·(√3; 1; √3)\)
Уравнение: \(√3·x + 1·y + √3·z + D = 0\)
Подставляем A₁: \(0 + 0 + √3·2 + D = 0 \Rightarrow D = -2√3\)
Уравнение: \(√3·x + y + √3·z - 2√3 = 0\)

3. Расстояние от точки A(0;0;0):
\(d = \dfrac{|0 + 0 + 0 - 2√3|}{√(3 + 1 + 3)} = \dfrac{2√3}{√7} = 2√\dfrac{3}{7}\)
