📖 Условие

При каком значении параметра \(a\) расстояние от точки \(A(3;a)\) до графика функции \(y = |x|\) будет минимальным? Найдите это расстояние.

✅ Ответ

-3

📖 Подробный разбор

1. График \(y = |x|\) состоит из двух лучей: \(y = x\) при \(x \ge 0\) и \(y = -x\) при \(x \le 0\)

2. Для луча \(y = x\) (\(x \ge 0\)):
Точка на луче \(B(t; t)\), \(t \ge 0\)
Квадрат расстояния: \(d_1^2(t) = (t-3)^2 + (t-a)^2 = 2t^2 - 2t(3+a) + 9 + a^2\)
Минимум при \(t = \dfrac{3+a}{2}\), если \(t \ge 0\)
\(d_1^2 = -\dfrac{(3+a)^2}{2} + 9 + a^2\)

3. Для луча \(y = -x\) (\(x \le 0\)):
Точка на луче \(C(t; -t)\), \(t \le 0\)
Квадрат расстояния: \(d_2^2(t) = (t-3)^2 + (-t-a)^2 = 2t^2 + t(2a-6) + 9 + a^2\)
Минимум при \(t = \dfrac{3-a}{2}\), если \(t \le 0\)
\(d_2^2 = -\dfrac{(3-a)^2}{2} + 9 + a^2\)

4. Рассмотрим \(f(a) = -\dfrac{(3-a)^2}{2} + 9 + a^2 = 0,5a^2 + 3a + 4,5\)

5. Минимум этой параболы при \(a = -\dfrac{3}{2\cdot0,5} = -3\)

6. При \(a = -3\) расстояние равно 0, так как точка \(A(3;-3)\) лежит на луче \(y = -x\)
