📖 Условие

Прямая \(y = 8x + 11\) является касательной к графику функции \(y = x³ + 5x² + 9x + 15\). Найдите абсциссу точки касания.

✅ Ответ

\(\dfrac{-5 + \sqrt{22}}{3}\)

📖 Подробный разбор

1. Производная функции: \(f'(x) = 3x² + 10x + 9\)

2. В точке касания угловой коэффициент прямой равен значению производной:
\(3x₀² + 10x₀ + 9 = 8\)

3. Решаем уравнение:
\(3x₀² + 10x₀ + 1 = 0\)
Дискриминант \(D = 100 - 12 = 88\)
\(x₀ = \dfrac{-10 \pm \sqrt{88}}{6} = \dfrac{-10 \pm 2\sqrt{22}}{6} = \dfrac{-5 \pm \sqrt{22}}{3}\)

4. Проверяем условие равенства значений:
Для \(x₀ = \dfrac{-5 + \sqrt{22}}{3}\) значение функции совпадает со значением касательной
