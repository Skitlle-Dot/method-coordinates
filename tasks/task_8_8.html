📖 Условие

Прямая \(y = 5x + 11\) является касательной к графику \(y = x³ + 4x² + 9x + 11\). Найдите абсциссу точки касания.

✅ Ответ

-2

📖 Подробный разбор

1. Производная функции: \(f'(x) = 3x² + 8x + 9\)

2. В точке касания угловой коэффициент прямой равен значению производной:
\(3x₀² + 8x₀ + 9 = 5\)

3. Решаем уравнение:
\(3x₀² + 8x₀ + 4 = 0\)
Дискриминант \(D = 64 - 4·3·4 = 64 - 48 = 16\)
\(x₀ = \dfrac{-8 \pm 4}{6}\)

4. Корни: \(x₁ = \dfrac{-8 + 4}{6} = -\dfrac{2}{3}\), \(x₂ = \dfrac{-8 - 4}{6} = -2\)

5. Проверяем условие равенства значений:
Для \(x₀ = -2\): \(f(-2) = -8 + 16 - 18 + 11 = 1\), \(5·(-2) + 11 = 1\) — подходит
Для \(x₀ = -\dfrac{2}{3}\): \(f(-\frac{2}{3}) = -\frac{8}{27} + \frac{16}{9} - 6 + 11 = \frac{430}{27}\), \(5·(-\frac{2}{3}) + 11 = \frac{23}{3} = \frac{207}{27}\) — не подходит
