📖 Условие

В прямоугольном параллелепипеде ABCDA₁B₁C₁D₁ известны ребра: AB = 2, AD = 3, AA₁ = 4. Найдите угол между плоскостями AB₁C и A₁B₁C.

✅ Ответ

\(\arccos \dfrac{5}{\sqrt{61}}\)

📖 Подробный разбор

1. Вводим систему координат: A(0;0;0), B(2;0;0), D(0;3;0), A₁(0;0;4), B₁(2;0;4), C(2;3;0)

2. Нормаль плоскости AB₁C через точки A(0;0;0), B₁(2;0;4), C(2;3;0):
\(\overrightarrow{AB₁} = (2; 0; 4)\)
\(\overrightarrow{AC} = (2; 3; 0)\)
\(\vec{n₁} = \overrightarrow{AB₁} × \overrightarrow{AC} = (-12; 8; 6) = 2·(-6; 4; 3)\)
Принимаем \(\vec{n₁} = (-6; 4; 3)\)

3. Нормаль плоскости A₁B₁C через точки A₁(0;0;4), B₁(2;0;4), C(2;3;0):
\(\overrightarrow{A₁B₁} = (2; 0; 0)\)
\(\overrightarrow{A₁C} = (2; 3; -4)\)
\(\vec{n₂} = \overrightarrow{A₁B₁} × \overrightarrow{A₁C} = (0; 8; 6) = 2·(0; 4; 3)\)
Принимаем \(\vec{n₂} = (0; 4; 3)\)

4. Косинус угла между плоскостями:
\(\cos φ = \dfrac{|\vec{n₁}·\vec{n₂}|}{|\vec{n₁}|·|\vec{n₂}|} = \dfrac{|(-6)·0 + 4·4 + 3·3|}{√(36 + 16 + 9)·√(0 + 16 + 9)} = \dfrac{25}{√61·5} = \dfrac{5}{√61}\)

5. Угол:
\(φ = \arccos \dfrac{5}{√61}\)
