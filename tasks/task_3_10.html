📖 Условие

В правильной шестиугольной призме ABCDEF A₁B₁C₁D₁E₁F₁ все ребра равны 1. Найдите угол между прямыми AB₁ и BC₁.

✅ Ответ

\(\arccos \dfrac{3}{4}\)

📖 Подробный разбор

1. Вводим систему координат: центр основания O(0;0;0)
Радиус описанной окружности R = 1
A(1; 0; 0), B(½; √3/2; 0), C(-½; √3/2; 0)
A₁(1; 0; 1), B₁(½; √3/2; 1), C₁(-½; √3/2; 1)

2. Находим координаты векторов:
\(\overrightarrow{AB₁} = (½ - 1; √3/2 - 0; 1 - 0) = (-½; √3/2; 1)\)
\(\overrightarrow{BC₁} = (-½ - ½; √3/2 - √3/2; 1 - 0) = (-1; 0; 1)\)

3. Скалярное произведение:
\(\overrightarrow{AB₁}·\overrightarrow{BC₁} = (-½)·(-1) + (√3/2)·0 + 1·1 = ½ + 0 + 1 = 1,5 = 3/2\)

4. Длины векторов:
\(|\overrightarrow{AB₁}| = √(¼ + ¾ + 1) = √2\)
\(|\overrightarrow{BC₁}| = √(1 + 0 + 1) = √2\)

5. Косинус угла:
\(\cos φ = \dfrac{3/2}{√2·√2} = \dfrac{3/2}{2} = \dfrac{3}{4}\)

6. Угол:
\(φ = \arccos \dfrac{3}{4}\)
