📖 Условие

В кубе ABCDA₁B₁C₁D₁ с ребром, равным 1, найдите расстояние от точки B до плоскости A₁C₁D.

✅ Ответ

\(\dfrac{2\sqrt{3}}{3}\)

📖 Подробный разбор

1. Вводим систему координат: A(0;0;0), B(1;0;0), C(1;1;0), D(0;1;0), A₁(0;0;1), C₁(1;1;1), D(0;1;0)

2. Составляем уравнение плоскости A₁C₁D через точки A₁(0;0;1), C₁(1;1;1), D(0;1;0):
\(\overrightarrow{A₁C₁} = (1; 1; 0)\)
\(\overrightarrow{A₁D} = (0; 1; -1)\)
Нормаль: \(\vec{n} = \overrightarrow{A₁C₁} × \overrightarrow{A₁D} = (-1; 1; 1)\)
Уравнение: \(-1·x + 1·y + 1·z + D = 0\)
Подставляем A₁: \(0 + 0 + 1 + D = 0 \Rightarrow D = -1\)
Уравнение: \(-x + y + z - 1 = 0\)

3. Расстояние от точки B(1;0;0):
\(d = \dfrac{|-1·1 + 0 + 0 - 1|}{√(1 + 1 + 1)} = \dfrac{|-2|}{√3} = \dfrac{2}{√3} = \dfrac{2√3}{3}\)
