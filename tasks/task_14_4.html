📖 Условие

\(В правильной четырёхугольной пирамиде SABCD сторона основания AB равна 4, а боковое ребро SA равно 7. На рёбрах CD и SC отмечены точки N и K соответственно, причём DN : NC = SK : KC = 1 : 3. Плоскость \alpha содержит прямую KN и параллельна прямой BC.\)
\(а) Докажите, что плоскость \alpha параллельна прямой SA.\)
\(б) Найдите угол между плоскостями \alpha и SBC.\)

✅ Ответ

\(\arccos\dfrac{37}{45}\)

📖 Подробный разбор

1. Вводим систему координат: B(0;0;0), A(4;0;0), C(0;4;0), D(4;4;0)
Центр основания O(2;2;0), высота SO = √(49 - 8) = √41, S(2;2;√41)

2. Находим координаты точек:
N(3; 4; 0)
K(1,5; 2,5; 3√41/4)

3. Строим плоскость α через K, N и точку P(3;0;0) (пересечение с AB)
Нормаль: \(\vec{n}_α = (3√41; 0; 6)\)

4. Вектор \(\overrightarrow{SA} = (2; -2; -√41)\). Проверяем параллельность:
\(\overrightarrow{SA}·\vec{n}_α = 2·3√41 + (-2)·0 + (-√41)·6 = 6√41 - 6√41 = 0\)
SA параллельна плоскости α

5. Нормаль к плоскости SBC: \(\vec{n}_{SBC} = (√41; 0; -2)\)

6. Косинус угла между плоскостями:
\(\cos φ = \dfrac{|3√41·√41 + 0·0 + 6·(-2)|}{√(9·41 + 36)·√(41 + 4)} = \dfrac{111}{135} = \dfrac{37}{45}\)
