📖 Условие

Прямая \(y = 6x + 7\) является касательной к графику функции \(y = 3x² + bx + 4\). Найдите \(b\), учитывая, что абсцисса точки касания больше 0.

✅ Ответ

нет решений

📖 Подробный разбор

1. Производная функции: \(f'(x) = 6x + b\)

2. В точке касания \(x₀\):
\(6x₀ + b = 6\) и \(3x₀² + bx₀ + 4 = 6x₀ + 7\)

3. Из первого уравнения: \(b = 6 - 6x₀\)

4. Подставляем во второе:
\(3x₀² + (6 - 6x₀)x₀ + 4 = 6x₀ + 7\)
\(3x₀² + 6x₀ - 6x₀² + 4 = 6x₀ + 7\)
\(-3x₀² + 6x₀ + 4 = 6x₀ + 7\)
\(-3x₀² = 3\)
\(x₀² = -1\)

5. Уравнение не имеет решений, значит прямая не является касательной
