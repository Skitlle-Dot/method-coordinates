📖 Условие

Из точки \(A(-2; 3)\) проведена касательная к параболе \(y = x^2 + 1\). Найдите точку касания.

✅ Ответ

\((-2 \pm \sqrt{2}; 7 \mp 4\sqrt{2})\)

📖 Подробный разбор

1. Пусть точка касания \(B(t; t^2 + 1)\)

2. Угловой коэффициент касательной к параболе в точке \(B\) равен производной:
\(k = 2t\)

3. Угловой коэффициент прямой \(AB\):
\(k_{AB} = \dfrac{(t^2 + 1) - 3}{t - (-2)} = \dfrac{t^2 - 2}{t + 2}\)

4. Приравниваем коэффициенты:
\(\dfrac{t^2 - 2}{t + 2} = 2t\)
\(t^2 - 2 = 2t(t + 2) = 2t^2 + 4t\)
\(0 = t^2 + 4t + 2\)

5. Решаем уравнение:
\(t = \dfrac{-4 \pm \sqrt{16 - 8}}{2} = \dfrac{-4 \pm 2\sqrt{2}}{2} = -2 \pm \sqrt{2}\)

6. Координаты точек касания:
Для \(t_1 = -2 + \sqrt{2}\):
\(x = -2 + \sqrt{2}\), \(y = (-2 + \sqrt{2})^2 + 1 = 4 - 4\sqrt{2} + 2 + 1 = 7 - 4\sqrt{2}\)
Для \(t_2 = -2 - \sqrt{2}\):
\(x = -2 - \sqrt{2}\), \(y = (-2 - \sqrt{2})^2 + 1 = 4 + 4\sqrt{2} + 2 + 1 = 7 + 4\sqrt{2}\)
