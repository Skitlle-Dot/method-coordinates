📖 Условие

Найдите абсциссу точки касания прямой \(y = 3x - 2\) и графика функции \(y = x² + x - 3\).

✅ Ответ

нет решений

📖 Подробный разбор

1. Производная функции: \(f'(x) = 2x + 1\)

2. В точке касания угловой коэффициент прямой равен значению производной:
\(2x₀ + 1 = 3\)
\(2x₀ = 2\)
\(x₀ = 1\)

3. Проверяем условие равенства значений:
\(f(1) = 1 + 1 - 3 = -1\)
\(3·1 - 2 = 1\) — не совпадает

4. Система не имеет решений, прямая не является касательной
