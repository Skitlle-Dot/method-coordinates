📖 Условие

Из пункта A по прямой дороге выехал велосипедист со скоростью 20 км/ч. Одновременно из пункта B, находящегося на расстоянии 15 км от пункта A перпендикулярно дороге, выехал мотоциклист со скоростью 40 км/ч по дороге, перпендикулярной первой. Найдите расстояние между велосипедистом и мотоциклистом через 0,5 часа.

✅ Ответ

\(5\sqrt{13}\) км

📖 Подробный разбор

1. Вводим систему координат: начало в A, ось X по дороге велосипедиста, ось Y перпендикулярно

2. Координаты B: \((0; 15)\)

3. Уравнения движения:
Велосипедист: \(x_1(t) = 20t\), \(y_1(t) = 0\)
Мотоциклист: \(x_2(t) = 40t\), \(y_2(t) = 15\)

4. Через 0,5 часа:
Велосипедист: \((10; 0)\)
Мотоциклист: \((20; 15)\)

5. Расстояние:
\(d = \sqrt{(10 - 20)^2 + (0 - 15)^2} = \sqrt{(-10)^2 + (-15)^2} = \sqrt{100 + 225} = \sqrt{325} = 5\sqrt{13} \approx 18,03\) км
