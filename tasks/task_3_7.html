📖 Условие

В правильной треугольной пирамиде SABC с основанием ABC известны ребра: AB = 6, SC = 5. Найдите угол между прямой SA и плоскостью SBC.

✅ Ответ

\(\arcsin \dfrac{3\sqrt{39}}{20}\)

📖 Подробный разбор

1. Вводим систему координат: центр основания O(0;0;0)
Радиус описанной окружности: R = 6/√3 = 2√3
A(-√3; 3; 0), B(2√3; 0; 0), C(-√3; -3; 0)
Высота: SO = √(25 - 12) = √13, S(0;0;√13)

2. Направляющий вектор SA:
\(\overrightarrow{SA} = (-√3; 3; -√13)\)

3. Находим нормаль к плоскости SBC:
\(\overrightarrow{SB} = (2√3; 0; -√13)\)
\(\overrightarrow{SC} = (-√3; -3; -√13)\)
\(\vec{n} = \overrightarrow{SB} × \overrightarrow{SC} = (-3√13; 3√39; -6√3)\)
Упрощаем: \(\vec{n} = (-√13; √39; -2√3)\)

4. Синус угла между прямой и плоскостью:
\(\sin φ = \dfrac{|\overrightarrow{SA}·\vec{n}|}{|\overrightarrow{SA}|·|\vec{n}|}\)

5. Скалярное произведение:
\(\overrightarrow{SA}·\vec{n} = (-√3)·(-√13) + 3·√39 + (-√13)·(-2√3) = √39 + 3√39 + 2√39 = 6√39\)

6. Длины:
\(|\overrightarrow{SA}| = √(3 + 9 + 13) = 5\)
\(|\vec{n}| = √(13 + 39 + 12) = √64 = 8\)

7. Синус:
\(\sin φ = \dfrac{6√39}{5·8} = \dfrac{3√39}{20}\)

8. Угол:
\(φ = \arcsin \dfrac{3√39}{20}\)
