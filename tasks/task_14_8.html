📖 Условие

В прямоугольном параллелепипеде ABCDA₁B₁C₁D₁ известны рёбра: AB = 4, BC = 3, CC₁ = 5. Найдите угол между плоскостями AB₁C и A₁B₁C.

✅ Ответ

\(\arccos\dfrac{4\sqrt{34}}{\sqrt{769}}\)

📖 Подробный разбор

1. Вводим систему координат: A(0;0;0), B(4;0;0), C(4;3;0), D(0;3;0), A₁(0;0;5), B₁(4;0;5), C₁(4;3;5)

2. Нормаль плоскости AB₁C:
\(\overrightarrow{AB₁} = (4; 0; 5)\)
\(\overrightarrow{AC} = (4; 3; 0)\)
\(\vec{n₁} = \overrightarrow{AB₁} × \overrightarrow{AC} = (-15; 20; 12)\)

3. Нормаль плоскости A₁B₁C:
\(\overrightarrow{A₁B₁} = (4; 0; 0)\)
\(\overrightarrow{A₁C} = (4; 3; -5)\)
\(\vec{n₂} = \overrightarrow{A₁B₁} × \overrightarrow{A₁C} = (0; 20; 12)\)

4. Косинус угла между плоскостями:
\(\cos φ = \dfrac{|\vec{n₁}·\vec{n₂}|}{|\vec{n₁}|·|\vec{n₂}|} = \dfrac{544}{√769·√544} = \dfrac{4√34}{√769}\)
