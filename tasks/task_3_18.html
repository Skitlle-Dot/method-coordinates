📖 Условие

В кубе ABCDA₁B₁C₁D₁ с ребром 2 найдите угол между плоскостями AB₁D₁ и BC₁D.

✅ Ответ

0

📖 Подробный разбор

1. Вводим систему координат: A(0;0;0), B(2;0;0), C(2;2;0), D(0;2;0), A₁(0;0;2), B₁(2;0;2), C₁(2;2;2), D₁(0;2;2)

2. Нормаль плоскости AB₁D₁:
\(\overrightarrow{AB₁} = (2; 0; 2)\)
\(\overrightarrow{AD₁} = (0; 2; 2)\)
\(\vec{n₁} = \overrightarrow{AB₁} × \overrightarrow{AD₁} = (-4; -4; 4) = 4·(-1; -1; 1)\)
Принимаем \(\vec{n₁} = (1; 1; -1)\)

3. Нормаль плоскости BC₁D:
\(\overrightarrow{BC₁} = (0; 2; 2)\)
\(\overrightarrow{BD} = (-2; 2; 0)\)
\(\vec{n₂} = \overrightarrow{BC₁} × \overrightarrow{BD} = (-4; -4; 4) = 4·(-1; -1; 1)\)
Принимаем \(\vec{n₂} = (1; 1; -1)\)

4. Векторы нормалей совпадают, значит плоскости параллельны, угол между ними равен 0°
