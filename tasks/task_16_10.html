📖 Условие

Две точки движутся по плоскости. Первая точка начинает движение из начала координат со скоростью (3; 4) м/с. Вторая точка начинает движение из точки (10; 5) со скоростью (-2; 1) м/с. Найдите минимальное расстояние между точками в процессе движения.

✅ Ответ

\(\dfrac{5\sqrt{34}}{34}\)

📖 Подробный разбор

1. Уравнения движения:
Точка 1: \(x_1(t) = 3t\), \(y_1(t) = 4t\)
Точка 2: \(x_2(t) = 10 - 2t\), \(y_2(t) = 5 + t\)

2. Квадрат расстояния:
\(\Delta x = 3t - 10 + 2t = 5t - 10\)
\(\Delta y = 4t - 5 - t = 3t - 5\)
\(d^2(t) = (5t - 10)^2 + (3t - 5)^2 = 34t^2 - 130t + 125\)

3. Вершина параболы при \(t_{min} = \dfrac{130}{68} = \dfrac{65}{34}\) с

4. Минимальное расстояние:
\(d_{min}^2 = 125 - \dfrac{4225}{34} = \dfrac{25}{34}\)
\(d_{min} = \dfrac{5}{\sqrt{34}} = \dfrac{5\sqrt{34}}{34}\)
