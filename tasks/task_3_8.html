📖 Условие

В правильной четырехугольной пирамиде SABCD все ребра равны 1. Найдите расстояние от точки A до плоскости SCD.

✅ Ответ

1

📖 Подробный разбор

1. Вводим систему координат: центр основания O(0;0;0)
Половина диагонали основания = √2/2
A(-√2/2; -√2/2; 0), B(√2/2; -√2/2; 0), C(√2/2; √2/2; 0), D(-√2/2; √2/2; 0)
Высота: SO = √(1 - ½) = 1/√2, S(0;0; 1/√2)

2. Составляем уравнение плоскости SCD через точки S, C, D:
\(\overrightarrow{SC} = (√2/2; √2/2; -1/√2)\)
\(\overrightarrow{SD} = (-√2/2; √2/2; -1/√2)\)
Нормаль: \(\vec{n} = \overrightarrow{SC} × \overrightarrow{SD} = (0; 2·(½); 2·(½)) = (0; 1; 1)\)
Уравнение: \(0·x + 1·y + 1·z + D = 0\)
Подставляем S: \(0 + 1/√2 + D = 0 \Rightarrow D = -1/√2\)
Уравнение: \(y + z - 1/√2 = 0\)

3. Расстояние от точки A:
\(d = \dfrac{|y_A + z_A - 1/√2|}{√(0² + 1² + 1²)} = \dfrac{| -√2/2 + 0 - 1/√2|}{√2}\)

4. Упрощаем: \(-√2/2 - 1/√2 = -√2/2 - √2/2 = -√2\)
Модуль: √2

5. Расстояние:
\(d = \dfrac{√2}{√2} = 1\)
