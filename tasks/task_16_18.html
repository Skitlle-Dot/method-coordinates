📖 Условие

Две точки движутся по окружности радиуса 1 с постоянными угловыми скоростями. Первая точка начинает движение из точки (1; 0) с угловой скоростью 1 рад/с. Вторая точка начинает движение из точки (-1; 0) с угловой скоростью 2 рад/с в том же направлении. Найдите момент времени \(t > 0\), когда расстояние между точками минимально.

✅ Ответ

\(\pi\) секунд

📖 Подробный разбор

1. Параметрические уравнения движения:
Первая точка: \(x_1(t) = \cos t\), \(y_1(t) = \sin t\)
Вторая точка: \(x_2(t) = -\cos(2t)\), \(y_2(t) = -\sin(2t)\)

2. Квадрат расстояния:
\(\Delta x = \cos t + \cos(2t)\)
\(\Delta y = \sin t + \sin(2t)\)
\(d^2(t) = (\cos t + \cos(2t))^2 + (\sin t + \sin(2t))^2\)

3. Используем формулы сложения:
\(d^2(t) = \cos^2 t + \cos^2(2t) + 2\cos t \cos(2t) + \sin^2 t + \sin^2(2t) + 2\sin t \sin(2t)\)
\(d^2(t) = (\cos^2 t + \sin^2 t) + (\cos^2(2t) + \sin^2(2t)) + 2(\cos t \cos(2t) + \sin t \sin(2t))\)
\(d^2(t) = 1 + 1 + 2\cos(t - 2t) = 2 + 2\cos(-t) = 2 + 2\cos t\)

4. Минимальное значение \(d^2(t)\) достигается при \(\cos t = -1\), то есть \(t = \pi\) с

5. Минимальное расстояние: \(d_{min} = \sqrt{2 + 2(-1)} = 0\) (точки встречаются)
