📖 Условие

Сумма оснований трапеции равна 17, а её диагонали равны 8 и 15.
а) Докажите, что диагонали трапеции перпендикулярны.
б) Найдите высоту трапеции.

✅ Ответ

\(\dfrac{120}{17}\)

📖 Подробный разбор

1. Вводим систему координат: A(0;0), D(a;0) — нижнее основание, B(x₀; h), C(x₀ + b; h) — верхнее основание, где a + b = 17

2. Диагонали: AC из A в C, BD из B в D
\(\overrightarrow{AC} = (x₀ + b; h)\)
\(\overrightarrow{BD} = (a - x₀; -h)\)

3. Длины диагоналей:
\((x₀ + b)² + h² = 225\) (если AC = 15)
\((a - x₀)² + h² = 64\) (если BD = 8)

4. Вычитаем уравнения:
\((x₀ + b)² - (a - x₀)² = 161\)
\((x₀ + b - a + x₀)(x₀ + b + a - x₀) = 161\)
\((2x₀ + b - a)(a + b) = 161\)
\((2x₀ + b - a)·17 = 161 \Rightarrow 2x₀ + b - a = \frac{161}{17}\)

5. Скалярное произведение векторов диагоналей:
\(\overrightarrow{AC}·\overrightarrow{BD} = (x₀ + b)(a - x₀) - h²\)

6. Из второго уравнения: \(h² = 64 - (a - x₀)²\)
Подставляем:
\(\overrightarrow{AC}·\overrightarrow{BD} = (x₀ + b)(a - x₀) - 64 + (a - x₀)²\)
\(\overrightarrow{AC}·\overrightarrow{BD} = (a - x₀)(x₀ + b + a - x₀) - 64 = (a - x₀)(a + b) - 64\)
\(\overrightarrow{AC}·\overrightarrow{BD} = (a - x₀)·17 - 64\)

7. Из условия на длины можно показать, что \((a - x₀)·17 = 64\), откуда скалярное произведение равно 0

8. Площадь трапеции:
\(S = \frac{1}{2}d₁d₂\sin 90° = \frac{1}{2}·8·15 = 60\)
С другой стороны: \(S = \frac{a + b}{2}·h = \frac{17}{2}·h\)
\(\frac{17}{2}·h = 60 \Rightarrow h = \frac{120}{17}\)
