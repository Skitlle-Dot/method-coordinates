📖 Условие

Даны векторы \(\vec{a}(-2; 4)\) и \(\vec{b}(2; -1)\). Известно, что векторы \(\vec{c}\) и \(\vec{b}\) сонаправлены, а \(|\vec{c}| = |\vec{a}|\). Найдите сумму координат вектора \(\vec{c}\).

✅ Ответ

2

📖 Подробный разбор

1. \(\vec{c}\) сонаправлен с \(\vec{b}\), значит \(\vec{c} = k·\vec{b}\) при k > 0:
\(\vec{c} = (2k; -k)\)

2. Длины векторов:
\(|\vec{a}| = √((-2)² + 4²) = √(4 + 16) = √20 = 2√5\)
\(|\vec{c}| = √((2k)² + (-k)²) = √(4k² + k²) = √(5k²) = k√5\)

3. Условие равенства длин:
\(k√5 = 2√5 \Rightarrow k = 2\)

4. Координаты \(\vec{c}\) и сумма:
\(\vec{c} = (4; -2)\)
\(4 + (-2) = 2\)
