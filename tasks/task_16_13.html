📖 Условие

Два автомобиля движутся по прямолинейному шоссе. Первый находится в точке с координатой 0 и движется со скоростью 40 км/ч в положительном направлении. Второй находится в точке с координатой 100 км и движется навстречу первому со скоростью 60 км/ч. Найдите момент времени, когда расстояние между автомобилями будет наименьшим.

✅ Ответ

1 час

📖 Подробный разбор

1. Вводим систему координат: ось X вдоль шоссе, начало в точке старта первого автомобиля

2. Уравнения движения:
Первый: \(x_1(t) = 40t\)
Второй: \(x_2(t) = 100 - 60t\)

3. Расстояние: \(d(t) = |40t - (100 - 60t)| = |100t - 100|\)

4. Минимальное расстояние при \(100t - 100 = 0\), то есть \(t = 1\) час

5. При \(t = 1\): \(x_1 = 40\) км, \(x_2 = 40\) км, \(d = 0\) (встретились)
