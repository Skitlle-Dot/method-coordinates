📖 Условие

В правильной треугольной пирамиде SABC с основанием ABC известны ребра: AB = 6, SC = 5. Найдите угол между прямой SA и плоскостью ABC.

✅ Ответ

\(\arcsin \dfrac{\sqrt{13}}{5}\)

📖 Подробный разбор

1. Вводим систему координат: центр основания O(0;0;0)
Радиус описанной окружности: R = 6/√3 = 2√3
A(-√3; 3; 0), B(2√3; 0; 0), C(-√3; -3; 0)
Высота: SO = √(25 - 12) = √13, S(0;0;√13)

2. Направляющий вектор SA:
\(\overrightarrow{SA} = (-√3; 3; -√13)\)

3. Нормаль к плоскости ABC (основание):
\(\vec{n} = (0; 0; 1)\)

4. Синус угла между прямой и плоскостью:
\(\sin φ = \dfrac{|\overrightarrow{SA}·\vec{n}|}{|\overrightarrow{SA}|·|\vec{n}|} = \dfrac{|(-√3)·0 + 3·0 + (-√13)·1|}{√(3 + 9 + 13)·1} = \dfrac{√13}{5}\)

5. Угол:
\(φ = \arcsin \dfrac{√13}{5}\)
