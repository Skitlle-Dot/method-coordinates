📖 Условие

Материальная точка движется по прямой \(x = t\), \(y = t\). Вторая точка находится в начале координат и начинает движение в тот же момент по оси \(X\) по закону \(x = t^2\). Найдите момент времени \(t > 0\), когда расстояние между точками будет минимальным.

✅ Ответ

0

📖 Подробный разбор

1. Первая точка \(M\): \((t; t)\). Вторая точка \(N\): \((t^2; 0)\)

2. Квадрат расстояния между \(M\) и \(N\):
\(d^2(t) = (t - t^2)^2 + (t - 0)^2 = t^2 - 2t^3 + t^4 + t^2 = t^4 - 2t^3 + 2t^2\)

3. Находим производную:
\(f'(t) = 4t^3 - 6t^2 + 4t = t(4t^2 - 6t + 4)\)

4. Дискриминант квадратного трехчлена \(4t^2 - 6t + 4\):
\(D = 36 - 64 = -28 < 0\), значит он всегда положителен

5. Единственная критическая точка \(t = 0\). При \(t > 0\) производная положительна, функция возрастает

6. Наименьшее расстояние достигается при \(t \to 0^+\)
