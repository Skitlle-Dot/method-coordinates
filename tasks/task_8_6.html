📖 Условие

Прямая \(y = 3x + 1\) является касательной к графику функции \(f(x) = ax² + 2x - 1\). Найдите значение \(a\).

✅ Ответ

-0,125

📖 Подробный разбор

1. Производная функции: \(f'(x) = 2ax + 2\)

2. В точке касания \(x₀\):
\(2ax₀ + 2 = 3\) и \(ax₀² + 2x₀ - 1 = 3x₀ + 1\)

3. Из первого уравнения: \(2ax₀ = 1 \Rightarrow ax₀ = 0,5\)

4. Второе уравнение упрощаем: \(ax₀² - x₀ - 2 = 0\)
Подставляем \(a = 0,5/x₀\):
\(\dfrac{0,5}{x₀}·x₀² - x₀ - 2 = 0\)
\(0,5x₀ - x₀ - 2 = 0\)
\(-0,5x₀ = 2\)
\(x₀ = -4\)

5. Находим \(a = \dfrac{0,5}{-4} = -0,125\)
