📖 Условие

В прямоугольном параллелепипеде ABCDA₁B₁C₁D₁ известны ребра: AB = 4, AD = 3, AA₁ = 5. Найдите угол между прямыми A₁C и BD.

✅ Ответ

\(\arccos \dfrac{7\sqrt{2}}{50}\)

📖 Подробный разбор

1. Вводим систему координат: A(0;0;0), B(4;0;0), D(0;3;0), A₁(0;0;5), C(4;3;0)

2. Находим координаты векторов:
\(\overrightarrow{A₁C} = (4-0; 3-0; 0-5) = (4; 3; -5)\)
\(\overrightarrow{BD} = (0-4; 3-0; 0-0) = (-4; 3; 0)\)

3. Скалярное произведение:
\(\overrightarrow{A₁C}·\overrightarrow{BD} = |4·(-4) + 3·3 + (-5)·0| = |-16 + 9| = 7\)

4. Длины векторов:
\(|\overrightarrow{A₁C}| = √(16 + 9 + 25) = √50 = 5√2\)
\(|\overrightarrow{BD}| = √(16 + 9 + 0) = √25 = 5\)

5. Косинус угла:
\(\cos φ = \dfrac{7}{5√2·5} = \dfrac{7}{25√2} = \dfrac{7√2}{50}\)

6. Угол:
\(φ = \arccos \dfrac{7√2}{50}\)
