📖 Условие

Прямая \(y = ax\) является касательной к графику функции \(y = x^{2} + 1\), причем абсцисса точки касания меньше нуля. Найдите значение \(a\).

✅ Ответ

-2

📖 Подробный разбор

1. Производная функции: \(f'(x) = 2x\)

2. В точке касания \(x₀\) выполняются условия:
\(a = 2x₀\) и \(x₀² + 1 = a·x₀\)

3. Подставляем первое во второе:
\(x₀² + 1 = 2x₀·x₀ = 2x₀²\)
\(x₀² = 1\)
\(x₀ = -1\) (по условию \(x₀ < 0\))

4. Находим \(a = 2·(-1) = -2\)
