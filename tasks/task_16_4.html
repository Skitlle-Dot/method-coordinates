📖 Условие

Из двух городов A и B, расстояние между которыми по прямой равно 300 км, одновременно навстречу друг другу вылетают два самолёта. Самолёт из A летит со скоростью 400 км/ч под углом 30° к линии AB, самолёт из B летит со скоростью 500 км/ч под углом 60° к линии AB (углы отсчитываются от линии AB в одну и ту же сторону). Найдите минимальное расстояние, на которое сблизятся самолёты.

✅ Ответ

300 км

📖 Подробный разбор

1. Вводим систему координат: начало в A, ось X направлена к B. Точка B имеет координату 300

2. Проекции скоростей:
Самолёт из A: \(v_{Ax} = 400\cos 30° = 200\sqrt{3}\), \(v_{Ay} = 400\sin 30° = 200\)
Самолёт из B: \(v_{Bx} = -500\cos 60° = -250\), \(v_{By} = 500\sin 60° = 250\sqrt{3}\)

3. Уравнения движения:
\(x_A(t) = 200\sqrt{3}t\), \(y_A(t) = 200t\)
\(x_B(t) = 300 - 250t\), \(y_B(t) = 250\sqrt{3}t\)

4. Квадрат расстояния:
\(\Delta x = 200\sqrt{3}t - 300 + 250t = 300 + t(250 + 200\sqrt{3})\)
\(\Delta y = 200t - 250\sqrt{3}t = t(200 - 250\sqrt{3})\)

5. После преобразований получаем \(d^2(t) = (a^2 + b^2)t^2 + 600a t + 90000\), где \(a = 250 + 200\sqrt{3}\), \(b = 200 - 250\sqrt{3}\)

6. Вершина параболы при отрицательном t, значит при \(t \ge 0\) функция возрастает, минимум при \(t = 0\)
