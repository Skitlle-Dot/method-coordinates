📖 Условие

В правильной четырехугольной призме ABCDA₁B₁C₁D₁ стороны основания равны 2, а боковые ребра равны 4. Найдите расстояние от точки B до плоскости A₁C₁D.

✅ Ответ

\(\dfrac{8}{3}\)

📖 Подробный разбор

1. Вводим систему координат: A(0;0;0), B(2;0;0), C(2;2;0), D(0;2;0), A₁(0;0;4), C₁(2;2;4), D(0;2;0)

2. Составляем уравнение плоскости A₁C₁D через точки A₁(0;0;4), C₁(2;2;4), D(0;2;0):
\(\overrightarrow{A₁C₁} = (2; 2; 0)\)
\(\overrightarrow{A₁D} = (0; 2; -4)\)
\(\vec{n} = \overrightarrow{A₁C₁} × \overrightarrow{A₁D} = (-8; 8; 4) = 4·(-2; 2; 1)\)
Уравнение: \(-2·x + 2·y + 1·z + D = 0\)
Подставляем A₁: \(0 + 0 + 4 + D = 0 \Rightarrow D = -4\)
Уравнение: \(-2x + 2y + z - 4 = 0\)

3. Расстояние от точки B(2;0;0):
\(d = \dfrac{|-2·2 + 2·0 + 0 - 4|}{√(4 + 4 + 1)} = \dfrac{|-4 - 4|}{√9} = \dfrac{8}{3}\)
