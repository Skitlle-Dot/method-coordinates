📖 Условие

Две лодки стартуют одновременно из одной точки на берегу озера и движутся прямолинейно с постоянными скоростями: первая под углом 60° к берегу, вторая под углом 120° к берегу (берег — прямая линия). Модули скоростей обеих лодок равны 10 км/ч. Через какое время после старта расстояние между лодками будет наименьшим? Найдите это расстояние.

✅ Ответ

0

📖 Подробный разбор

1. Вводим систему координат: начало в точке старта, ось X вдоль берега, ось Y перпендикулярно берегу в сторону воды

2. Проекции скоростей:
Первая лодка: \(v_{1x} = 10\cos 60° = 5\), \(v_{1y} = 10\sin 60° = 5\sqrt{3}\)
Вторая лодка: \(v_{2x} = 10\cos 120° = -5\), \(v_{2y} = 10\sin 120° = 5\sqrt{3}\)

3. Уравнения движения:
\(x_1(t) = 5t\), \(y_1(t) = 5\sqrt{3}t\)
\(x_2(t) = -5t\), \(y_2(t) = 5\sqrt{3}t\)

4. Квадрат расстояния:
\(d^2(t) = (5t + 5t)^2 + (5\sqrt{3}t - 5\sqrt{3}t)^2 = (10t)^2 = 100t^2\)

5. Функция возрастает при \(t > 0\), минимум при \(t = 0\)
