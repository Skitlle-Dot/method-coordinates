### Задача 1.1
**Условие:** Найдите площадь квадрата, вершины которого имеют координаты (9; 0), (10; 9), (1; 10), (0; 1).

**Решение:**
1. Находим длину стороны AB: A(9;0), B(10;9)
   AB = √((10-9)² + (9-0)²) = √(1 + 81) = √82

2. Площадь квадрата: S = AB² = 82

**Ответ:** 82

2. Второе неравенство — область выше угла \(y = |x - a|\) с вершиной \((a;0)\)

3. Единственное решение при касании стороны угла с параболой

4. Для стороны \(y = x - a\): \(x - a = 2 - x^2 \Rightarrow x^2 + x - (a + 2) = 0\)
   Дискриминант \(1 + 4(a + 2) = 0 \Rightarrow a = -\dfrac{9}{4}\)

5. Для стороны \(y = -x + a\): \(-x + a = 2 - x^2 \Rightarrow x^2 - x + (a - 2) = 0\)
   Дискриминант \(1 - 4(a - 2) = 0 \Rightarrow a = \dfrac{9}{4}\)

**Ответ:** \(-\dfrac{9}{4}; \dfrac{9}{4}\)

**Решение:**

### Задача 17.1
**Условие:** Сумма оснований трапеции равна 17, а её диагонали равны 8 и 15.
а) Докажите, что диагонали трапеции перпендикулярны.
б) Найдите высоту трапеции.

**Решение:**
1. Вводим систему координат: A(0;0), D(a;0) — нижнее основание, B(x₀; h), C(x₀ + b; h) — верхнее основание, где a + b = 17

2. Диагонали: AC из A в C, BD из B в D
   \(\overrightarrow{AC} = (x₀ + b; h)\)
   \(\overrightarrow{BD} = (a - x₀; -h)\)

3. Длины диагоналей:
   \((x₀ + b)² + h² = 225\) (если AC = 15)
   \((a - x₀)² + h² = 64\) (если BD = 8)

4. Вычитаем уравнения:
   \((x₀ + b)² - (a - x₀)² = 161\)
   \((x₀ + b - a + x₀)(x₀ + b + a - x₀) = 161\)
   \((2x₀ + b - a)(a + b) = 161\)
   \((2x₀ + b - a)·17 = 161 \Rightarrow 2x₀ + b - a = \frac{161}{17}\)

5. Скалярное произведение векторов диагоналей:
   \(\overrightarrow{AC}·\overrightarrow{BD} = (x₀ + b)(a - x₀) - h²\)

6. Из второго уравнения: \(h² = 64 - (a - x₀)²\)
   Подставляем:
   \(\overrightarrow{AC}·\overrightarrow{BD} = (x₀ + b)(a - x₀) - 64 + (a - x₀)²\)
   \(\overrightarrow{AC}·\overrightarrow{BD} = (a - x₀)(x₀ + b + a - x₀) - 64 = (a - x₀)(a + b) - 64\)
   \(\overrightarrow{AC}·\overrightarrow{BD} = (a - x₀)·17 - 64\)

7. Из условия на длины можно показать, что \((a - x₀)·17 = 64\), откуда скалярное произведение равно 0

8. Площадь трапеции:
   \(S = \frac{1}{2}d₁d₂\sin 90° = \frac{1}{2}·8·15 = 60\)
   С другой стороны: \(S = \frac{a + b}{2}·h = \frac{17}{2}·h\)
   \(\frac{17}{2}·h = 60 \Rightarrow h = \frac{120}{17}\)

**Ответ:** \(\dfrac{120}{17}\)

7. Косинус угла BKM:
   \(\cos\angle BKM = \dfrac{3a²/10}{(a\sqrt{10}/5)·(3a\sqrt{5}/10)} = \dfrac{1}{\sqrt{2}}\)
   Следовательно, \(\angle BKM = 45°\)

8. Для нахождения радиуса окружности, описанной около треугольника ABK:
   A(0;0), B(a;0), K(4a/5; 3a/5)
   Стороны: AB = a, AK = a, BK = a√10/5

9. Треугольник ABK равнобедренный (AB = AK)
   Площадь: \(S = \frac{1}{2}·AB·h = \frac{1}{2}·a·\frac{3a}{5} = \frac{3a²}{10} = \frac{3·(4√5)²}{10} = \frac{3·80}{10} = 24\)

10. Радиус описанной окружности:
    \(R = \dfrac{AB·AK·BK}{4S} = \dfrac{a·a·(a√10/5)}{4·24} = \dfrac{a³√10/5}{96} = \dfrac{(4√5)³√10}{480} = \dfrac{64·5√5·√10}{480} = \dfrac{64·5√50}{480} = \dfrac{64·5·5√2}{480} = \dfrac{1600√2}{480} = \dfrac{10√2}{3}\)

**Ответ:** \(\dfrac{10\sqrt{2}}{3}\)

### Задача 17.5
**Условие:** В параллелограмме ABCD с острым углом BAD из вершины B проведены высоты BP и BQ, причём точка P лежит на стороне AD, а точка Q — на стороне CD. На стороне AD отмечена точка M. Известно, что AM = BP, AB = BQ.
а) Докажите, что BM = PQ.
б) Найдите площадь треугольника APQ, если AM = BP = 21, AB = BQ = 29.

**Решение:**
1. Вводим систему координат: A(0;0), B(29;0), D(p;q), C(29 + p; q), где q > 0

2. Высота BP на AD. Расстояние от B до прямой AD:
   Уравнение AD: qx - py = 0
   \(BP = \dfrac{|q·29 - p·0|}{\sqrt{p² + q²}} = \dfrac{29q}{\sqrt{p² + q²}} = 21\)

3. Высота BQ на CD. CD параллельна AB, значит её уравнение: y = q
   Тогда Q = (29; q), и BQ = q = 29

4. Из условия BP = 21 и q = 29:
   \(\dfrac{29·29}{\sqrt{p² + 29²}} = 21 \Rightarrow \sqrt{p² + 29²} = \dfrac{841}{21}\)
   \(p² + 841 = \dfrac{841²}{441} \Rightarrow p² = 841·\left(\dfrac{841}{441} - 1\right) = 841·\dfrac{400}{441}\)
   \(p = \dfrac{29·20}{21} = \dfrac{580}{21}\)

5. Точка M на AD, AM = BP = 21. Длина AD = √(p² + q²) = 841/21
   Параметр t = AM/AD = 21 / (841/21) = 441/841
   M = (t·p; t·q) = \(\left(\dfrac{441}{841}·\dfrac{580}{21}; \dfrac{441}{841}·29\right) = \left(\dfrac{12180}{841}; \dfrac{441}{29}\right)\)

6. Точка P — проекция B на AD. Координаты P находим из условия, что P лежит на AD и BP ⟂ AD
   После вычислений получаем координаты P

7. Доказываем, что BM = PQ, проверяя равенство расстояний

8. Площадь треугольника APQ:
   A(0;0), Q(29;29), P с найденными координатами
   Вычисляем определитель:
   \(\overrightarrow{AP} = \left(\frac{29·580²}{841²}; \frac{29²·580}{841²}\right) = \frac{29·580}{841²}·(580; 29)\)
   \(\overrightarrow{AQ} = (29; 29)\)

   Площадь \(S = \frac{1}{2}|\det(\overrightarrow{AP}, \overrightarrow{AQ})| = \frac{1}{2}·\frac{29·580}{841²}·|580·29 - 29·29| = \frac{1}{2}·\frac{29·580}{841²}·29·551\)

   Учитывая, что 841 = 29², получаем \(S = \frac{290·551}{841} = \frac{290·19·29}{841} = \frac{290·19·29}{29²} = \frac{290·19}{29} = 10·19 = 190\)

**Ответ:** 190
